## How to compile and run
需要先安裝pygame的相關套件, 之後弄比較完整會補上轉好的exe及mac的app檔
```sh
pip install -r requirements.txt
```

執行方式: ``` python menu.py ```

## 操作方式
左邊玩家： wsad left-shift  
右邊玩家： 上下左右 space
